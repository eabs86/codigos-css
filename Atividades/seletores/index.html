Seletores CSS

Segue um resumo sobre os tipos de seletores em CSS

Tipos de Seletores:

  • Seletor por Tipo(ou Tag): Busca por uma tag HTML específica
  • Seletor por ID(#): Busca por elementos através do atributo ID
  • Seletor por Classe(.): Busca por elemento através do atributo "class"
  • Seletor Universal(*): Selectiona todos os elementos HTML
  • Seletor de Atributo([atrib]): Selectiona elementos que possuem um atributo específico em nosso documento HTML. Conseguimos buscar também por atributos com valor específico.
