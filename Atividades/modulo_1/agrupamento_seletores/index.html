Estudando agrupamento de seletores

Pode-se aplicar os mesmos estilos CSS para vários seletores diferentes.

Para isso, pode-se separar os seletores utilizando ","

Dessa forma, não é preciso ficar copiando e colando os mesmos códigos para todos. Pode-se fazer um bloco só de CSS e reutilizar as mesmas regras.
