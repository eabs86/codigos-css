Trabalhando com Medidas Absolutas.

São elas: cm, mm, Q(quarto de milímetro), in (polegadas), pc(Paica), pt(pontos), px(pixels)

Algumas unidades não fazem sentido serem utilizadas quando se estar utilizando tela - ex: centímetros. Geralmente para telas usa-se pixels.

Diferença entre Pixel Físico e Pixel Lógico:

  • Fisico: Pixels que realmente existem no dispositivo
  • Lógico: Pixel renderizado. Quanto de conteúdo cabe na tela
