Unidade de Medida "em"

Utiliza como referência a propriedade do font-size

font-size: 1em = 16px

font-size: 2em = 32px

font-size: 3em = 64px




Unidade de Medida "rem"

Utiliza como referência a propriedade do elemento raiz ( por isso rem = root em) O elemento raiz de uma página é a tag html É melhor usar o rem pois ele fica atrelado e ajustado automaticamente ao tamanho da fonte configurada no browser.

font-size: 1em = 16px

font-size: 2em = 32px

font-size: 3em = 64px




Unidade de Medida "vw"

Viewport é a área visível de uma página web. Pode varia de dispositivo para dispositivo. 1vw = 1% do tamanho da largura da área visível da página
